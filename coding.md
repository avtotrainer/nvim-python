### РЉА рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ

рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљрЃерЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃ«рЃЊрЃЮрЃЉрЃљ <C-w> рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА, рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃЏрЃљ рЃљрЃЏрЃљрЃю рЃарЃЮрЃЏ рЃўрЃЏрЃБрЃерЃљрЃЮрЃЋрЃЮрЃА, рЃарЃЕрЃћрЃЋрЃљ: 

nvim рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃћрЃЌ рЃљрЃЏ [рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃЊрЃљрЃю](https://github.com/avtotrainer/nvchad-2.5-config)

## РЉб рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ (рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃљрЃџрЃБрЃарЃў)

```vim
<C-\>
```

## РЉБ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃљ

рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў рЃњрЃљрЃЊрЃљрЃЋрЃЊрЃўрЃЋрЃљрЃарЃЌ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃ«рЃАрЃюрЃћрЃЉрЃљ Terminal рЃарЃћрЃЪрЃўрЃЏрЃерЃў. рЃЌрЃБ рЃљрЃарЃљ:

```vim
i
```

## РЉц рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

## РЉЦ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃўрЃЊрЃљрЃю рЃњрЃљрЃЏрЃЮрЃАрЃЋрЃџрЃљ рЃЊрЃљ рЃЎрЃЮрЃЊрЃќрЃћ рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљ

рЃЌрЃБ рЃњрЃЋрЃўрЃюрЃЊрЃљ рЃбрЃћрЃарЃўрЃюрЃљрЃџрЃў рЃЊрЃљрЃўрЃ«рЃБрЃарЃЮрЃА, рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃўрЃњрЃўрЃЋрЃћ рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃљрЃА <C-\> рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЋрЃЉрЃарЃБрЃюрЃЊрЃћрЃЉрЃўрЃЌ рЃцрЃљрЃўрЃџрЃўрЃА рЃЉрЃБрЃцрЃћрЃарЃерЃў

рЃАрЃ«рЃЋрЃљрЃюрЃљрЃўрЃарЃљрЃЊ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў рЃњрЃљрЃЊрЃљрЃЋрЃЊрЃўрЃЋрЃљрЃарЃЌ NTermnal (Normal) рЃарЃћрЃЪрЃўрЃЏрЃерЃў рЃЊрЃљ рЃЋрЃўрЃюрЃљрЃфрЃЋрЃџрЃћрЃЉрЃЌ рЃцрЃљрЃюрЃ»рЃарЃўрЃЌ рЃЏрЃљрЃдрЃџрЃљ
```vim
 <Esc>
<C-k>
```
